HTML Struktūra

HTML tai kompiuterinė kalba skirta struktūrizuoti žiniatinklio turinį.
Dauguma elementų turi atidarymo ir uždarymo "tag"us. Viskas kas yra tarp atidarymo ir uždarymo traktuojama kaip vaikiniai elementai arba turinys.
Bet koks HTML elementas gali turėti tik vieną tėvą ir betkokį kiekį vaikų.
